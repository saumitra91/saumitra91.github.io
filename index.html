Task Manager API

This is a node backend having multiple endpoints with authentication.

They are:

  • /users - POST, creates a new user, returns a token.

    • Request body is : { "name":"John Doe", "email":"John.Doe@mail-server.com", "password":"password" }

  • /users/login - POST, logs in a user, returns a token.

    • Request body is : { "email":"John.Doe@mail-server.com", "password":"password" }

  • /users/logout - POST - logs out the currently authenticated user, uses the token from above in Authorization headers.
  • /users/me - GET, PATCH, DELETE, read or update or delete the current user, using token.
  • /users/me/avatar - POST, DELETE - upload a picture.

    • Request body is form-data with key='avatar' and value as profile-picture.jpg

  • /tasks - POST, GET - reads tasks for an authenticated user.

    • Request body is : { "description":"task description", "completed":true }

  • /tasks/:id - GET, PATCH, DELETE - read or update or delete single task.
